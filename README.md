# Spotify Redesign

Une refonte moderne de l'application Spotify avec des fonctionnalités sociales améliorées.

## Fonctionnalités
- Écoute en direct (Live Party)
- Playlists collaboratives
- Memory Playlists
- Audio Fingerprint
- Système de badges

## Technologies utilisées
- React
- Context API
- React Router

## Installation
```bash
npm install
npm start
